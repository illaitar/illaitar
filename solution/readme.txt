----------------------------------------------------------------------------------------------------
                                            Реализация
----------------------------------------------------------------------------------------------------

 Основной класс SparseMatrix описан в файлах SparseMatrix.cpp и SparseMatrix.hpp. Вспомогательные классы для адресной арифметики также описаны в этих файлах

 Хранение происходит в формате CSR.

 Реализованы:

● double get(size_t, size_t) - получение значения по индексам (поддержка константных объектов)

● void set(size_t, size_t, double) - изменение значения по индексам

● size_t num_rows() - вывод количества строк (поддержка константных объектов)

● size_t num_columns() - вывод количества столбцов (поддержка константных объектов)

● операции сравнения двух матриц (==, !=) (поддержка константных объектов)
 Точность сравнения задана изначально, но пользователь может ее поменять через функцию set_epsilon(double)

● бинарная операция сложения матриц + (поддержка константных объектов)

● бинарная операция умножения матриц * (поддержка константных объектов)

● операция индексации [i][j] (поддержка константных объектов)

● операции для поддержки адресной арифметики (поддержка константных объектов)

● Фабричная функция MakeOnes(size_t, size_t) - создание единичной матрицы заданного размера

----------------------------------------------------------------------------------------------------
                                            Makefile
----------------------------------------------------------------------------------------------------

Команды сборки:

● make / make all - создание файла SparseMatrix.o, исполняемого файла prog, запуск ./prog

● make SparseMatrix.o - создание файла SparseMatrix.o

● make tests - последовательные компиляция и запуск 7 тестов с проверкой покрытия и выводом в html

● make clean - очистка всего временного

----------------------------------------------------------------------------------------------------
